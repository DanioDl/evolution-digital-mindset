# Evolution Digital Mindset

Статический портал для GitHub Pages.

## Как выложить

1. Распакуйте архив.
2. Откройте папку `evolution-digital-mindset`.
3. Загрузите в корень репозитория именно содержимое папки: `index.html`, `ai/`, `ba/`, `assets/`, `community/`, `library/`, `practices/`, `results/`, `README.md`.
4. GitHub → Settings → Pages → Deploy from a branch → `main` → `/root`.
5. Сайт будет доступен по адресу вида: `https://<логин>.github.io/evolution-digital-mindset/`.

## Структура

- `index.html` — главная страница
- `ai/` — раздел ИИ
- `ba/` — раздел бизнес-анализа с процессами, BPMN, требованиями, фреймами и шаблонами
- `assets/css/style.css` — единый стиль портала
- `assets/js/app.js` — общий JavaScript
